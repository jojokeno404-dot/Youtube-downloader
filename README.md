# YouTube Downloader

A simple Windows desktop application built with Python and Tkinter
for downloading YouTube videos and audio.

## Features
- Download single videos (MP4)
- Download audio only (MP3)
- Progress bar
- FFmpeg support
- Standalone Windows executable

## How to Use
1. Download the `.exe` from the Releases section
2. Run the program
3. Paste a YouTube video URL
4. Choose download format

## Requirements (for source code)
- Python 3.10+
- yt-dlp
- FFmpeg
